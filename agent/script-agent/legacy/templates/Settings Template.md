# Settings Template (设定集)

> **定位**：项目全局资产注册表。所有在后续「分集大纲」与「分集剧本」中出现的人物、物品、场景**必须**先在本设定集中以 `∆分类 + @名称` 格式注册，方可在下游引用。
>
> **资产标记体系（硬规则）**：
>
> - `∆` 为**资产分类标记**，仅三种：`∆人物`、`∆物品`、`∆场景`。
> - `@` 为**具体资产名称标记**。**人物、物品、场景三类统一**，一律使用全角括号格式：
>   **`@主称谓（中文名）`**
>   - **主称谓**：人物为剧本/字幕常用英文名或主名（**可含空格**）；物品、场景可为英文道具名、地点英文名或项目内主称谓（与系列圣经、STAGE 2 一致）。
>   - **括号**：须为 **全角** `（` `）`，**禁止**用半角 `()` 代替。
>   - **括号内**：须为 **中文** 全名或官方中文称谓（与人物小传、圣经、地图标注一致），**禁止省略括号**、禁止括号内再写英文顶替中文。
>   - **禁止**只写 `@Claire Hart` 或 `@Claire Hart(Claire Hart)` 等无中文、括号错误的形式。
> - 名称一经注册，后续 STAGE 6/7 **每次**引用**必须**与本行 `@` 后整段**完全一致**（含空格与全角括号及中文），一字不差。
> - 新资产须先在本设定集中追加注册，再在大纲/剧本中引用。
>
> **来源**：本设定集的初始草稿可由 LLM 从 STAGE 1-4 产物中自动提取，用户审核补充后确认。
