# Characters Template

> **市场与人设默认**：面向**海外女性向 BL 短剧**；角色姓名、背景与口语节奏**默认海外本地化**（非华裔为常态）。**仅当**主创明确要求「中国人 / 华裔 / 国内都市」等，再按指定范围书写。详见 `agent/script-agent/prompts/rule.md` §6.1。
>
> **分步交付（同一 STAGE 2）**：第一轮优先只写下方 **## 核心关系定义** + **## 人物矩阵总览**；确认后再分别输出 **## 主角一**、**## 主角二**（默认一轮一人）；最后分批输出 **## 配角一**、**## 配角二**……（每人一节 `##`，与双男主同级，勿仅用姓名起段）。详见 `agent/script-agent/prompts/flowchart.md` §2.4。
>
> **配角节数**：须展开小传的配角人数 **= 人物矩阵中需写满的配角行（+ 阶段性人物若也要小传）**，**不等于**本模板下文印了几段；矩阵里列了几名要写，就重复 **## 配角N** 若干节，分批交付时每轮仍建议 1～3 人。
>
> **禁止**在交稿中使用未替换的占位符 `[姓名]`；须写**具体姓名或可追踪代号**。
>
> **与 STAGE 5 设定集衔接**：进入设定集注册资产时，人物在全局表中须写为 `**@主称谓（中文全名）`**（全角括号、括号内中文），与英文名/主名并列展示；详见 `agent/script-agent/templates/Settings Template.md`。
