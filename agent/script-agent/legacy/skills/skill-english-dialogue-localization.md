# Skill：英语对白本地化（Locale Brief + STAGE 7）

## 触发条件

- `maxApprovedStage >= 6` 且即将或正在输出 **STAGE 7** 分集剧本；或
- 主创明确要求「英语更地道 / 去中式英语 / 按某地区口语改对白」。

## 输入（须齐备再写英语台词）

- `Project.meta`：`targetMarket`、`dialogueLanguage`、剧名等。
- 立项摘要 / `creativeBrief`、**系列圣经**中与 **地域、时代、阶层、职业场景** 相关的段落。
- **STAGE 5 设定集**：资产为 **`@主称谓（中文名）`** 全串（人物/场景/物品同一格式）；写对白时 `@` 引用须与设定集**逐字一致**。
- **侧栏《英语 Locale 简报》**（`englishLocaleBrief`）：全剧一份 SSOT；若为空，**须先在 Studio 点击「生成/更新英语 Locale 简报」**，不得凭空虚构「当地人都这么说」。

## 步骤

1. **确定 Primary locale**  
   从 meta 与圣经归纳：例如 *General American*、*British English (England)*、*某城市口语层*；若项目明确「美区 / 欧区 / 澳新」等，与简报第一节对齐。

2. **读取或生成 Locale 简报**  
   - 若侧栏已有简报：本轮以它为唯一语体依据，**不**为省 token 重复泛泛检索。  
   - 若缺简报：**停止写正文**，提示主创在编剧室侧栏生成简报后再写 STAGE 7。

3. **写 STAGE 7 英语台词时**（格式仍遵守分集模板：句前括号表演、**英语 + 句末（中文翻译）**、说话人须为 **`@主称谓（中文名）`** 与设定集一致）  
   - 对照简报中的：**称谓、敬语/粗鲁度层级、省略与缩略、习语与感叹**、**职业/场景高频说法**。  
   - **避免**：中文语序直译、叠床架屋的礼貌套话、书面语演讲腔、错误习语、与人物教育背景不符的「过于文雅或过于俚」。

4. **自检（静默）**  
   - 每条英语是否可读作 **母语者在该设定下** 会说的话？  
   - 是否出现典型 Chinglish（见 `agent/script-agent/prompts/english-lines-agent-role.md`）？

## 《英语 Locale 简报》输出模板（Markdown，固定小节）

工程或人工整理时建议使用以下结构（便于侧栏编辑与检索）：

```markdown
## 地域与语体
- Primary locale：（如 US / UK / 其他）
- 时代与社会层：（如当代都市中产 / 校园 / 职场）
- 语体要点：（正式↔随意、缩写偏好等）

## 称谓与礼貌层级
- 人物间如何称呼（名/姓/昵称/职衔）
- 冲突/亲密场景下的语气变化

## 高频口语与避免 Chinglish
- 推荐句式或口头禅类型（概括，勿堆砌）
- 明确禁止的直译式表达（若有）

## 文化敏感与禁忌（创作向）
- 创作时需避让的冒犯性称谓或话题（仅作剧本分寸参考）

## 检索来源附录
- [标题](URL)（可选，便于主创纠偏）

## 本剧专用词汇表（可选）
- `@主称谓（中文名）`：习惯用语或绰号（须与 STAGE 5 注册全串一致）
```

- 全剧 **一份** Locale 简报已写入项目并在 STAGE 7 工程注入中可见节录。  
- STAGE 7 正文台词可逐条对照简报做语体自检；无明显 Chinglish 与地域违和。
