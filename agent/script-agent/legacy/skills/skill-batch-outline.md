# skill-batch-outline：5～10 集粗纲批量产出

## 触发条件

- 需要连续规划多集（粗分场或段落级），而非直接写满正文。

## 输入

- 起始集数与结束集数（建议一次 **5～10 集**）。
- 当前里程碑进度与下一阶段目标（来自 `03_SERIES_BIBLE.md` 或主创说明）。
- 上一批最后一集的集尾卡点（保证钩子链不断）。

## 步骤

1. 列出本批 **每集一句 logline**（含关系变化方向）。
2. 为每集标注 **集尾卡点类型**，用 `agent/script-agent/knowledge/04_HOOK_LEXICON.md` 做**轮换检查**（避免连续同型）。
3. 标注每集 **新信息点数量**（建议 0～2），超标则拆移到他集。
4. 粗分场：每集 **2～4 场**一句话说明（不必写全对白）。
5. 检查 **第 1 集开场**是否与上一批末集 **3 秒接戏** 规则一致（`05_CLIFFHANGER_RULES.md`）。

## 引用 knowledge

- `agent/script-agent/knowledge/02_SHORTFORM_PACING.md`
- `agent/script-agent/knowledge/03_SERIES_BIBLE.md`
- `agent/script-agent/knowledge/04_HOOK_LEXICON.md`
- `agent/script-agent/knowledge/05_CLIFFHANGER_RULES.md`

## 输出格式

- 表格或分级列表：**集数 | 一句 logline | 集尾类型 | 场次数 | 新信息点**。
- 可选附录：本批内 **钩子链** 一句话串联。

## 完成标准

- 每集有独立推进点；集尾类型无连续 3 集同型；信息投放曲线无单集过载。
