# 内容合规自检清单

## 核心理念：红线不能碰，灰区要小心

微短剧审核趋严，一旦触碰红线，整部剧可能下架甚至连累平台账号。合规不是创作的敌人，而是护栏——在护栏内把故事讲好，才是真本事。

**合规三原则：**

1. **红线原则：** 绝对禁止的内容，零容忍
2. **灰区原则：** 敏感但可处理的内容，需要技巧性规避
3. **正向原则：** 主动传递积极价值观，不只是"不犯错"

## 一、红线清单（绝对禁止）

### 1. 政治类红线


| 禁止内容   | 说明               |
| ------ | ---------------- |
| 损害国家形象 | 丑化国家机关、公职人员形象    |
| 歪曲历史事实 | 篡改重大历史事件、抹黑英雄人物  |
| 危害国家统一 | 涉及领土主权、民族分裂相关内容  |
| 煽动对立情绪 | 挑拨民族关系、地域歧视、阶层对立 |
| 违反宗教政策 | 宣扬宗教极端、邪教内容      |


### 2. 暴力类红线


| 禁止内容   | 说明                  |
| ------ | ------------------- |
| 血腥暴力细节 | 特写伤口、血液喷射、肢体分离等     |
| 虐待展示   | 详细展示虐待过程（包括动物虐待）    |
| 教唆犯罪   | 展示犯罪手法细节、提供可模仿的操作步骤 |
| 美化暴力   | 将暴力行为塑造为"酷"或"正义"    |
| 校园暴力美化 | 将校园欺凌浪漫化或轻描淡写       |


### 3. 色情类红线


| 禁止内容   | 说明                |
| ------ | ----------------- |
| 裸露/性行为 | 任何程度的裸露或性行为画面     |
| 性暗示过度  | 过分露骨的性暗示镜头或台词     |
| 未成年相关  | 任何涉及未成年人的情色内容     |
| 性骚扰美化  | 将性骚扰行为塑造为"霸道""浪漫" |
| 物化身体   | 以展示身体为目的的无剧情镜头    |


### 4. 违法类红线


| 禁止内容 | 说明             |
| ---- | -------------- |
| 涉毒内容 | 展示吸毒过程、美化毒品    |
| 赌博教唆 | 展示赌博技巧、美化赌博    |
| 诈骗手法 | 详细展示诈骗流程和技巧    |
| 非法集资 | 宣传传销、非法金融活动    |
| 侵犯隐私 | 偷拍、跟踪等行为作为正面手段 |

## 二、高风险灰区（需要技巧性处理）

### 1. 价值观灰区


| 灰区内容     | 风险点          | 处理方式                      |
| -------- | ------------ | ------------------------- |
| 主角"法外开恩" | 主角以暴制暴、私刑正义  | 必须有法律后果或角色反思，不能把私刑当正义     |
| 金钱万能论    | 用钱解决一切问题     | 核心冲突要靠智慧/情感/正当手段解决，金钱只是辅助 |
| 封建糟粕     | 包办婚姻、重男轻女当正面 | 可以展示但必须批判，不能美化            |
| 拜金主义     | 嫌贫爱富、唯金钱论    | 反派可以拜金，主角必须展示正确价值观        |
| 仇富/仇穷    | 刻板化描写某个阶层    | 角色可以有阶层差异，但不能全面丑化某个群体     |
| 职场PUA美化  | 将职场压迫美化为"磨练" | 主角应反抗不合理待遇，不是逆来顺受         |


### 2. 关系类灰区


| 灰区内容   | 风险点        | 处理方式                       |
| ------ | ---------- | -------------------------- |
| 霸道总裁强吻 | 未经同意的亲密行为  | 改为双向情感确认后的亲密，或用"差点亲上"替代    |
| 年龄差恋爱  | 年龄差距过大引发争议 | 确保双方都是成年人，避免暗示权力不对等        |
| 三角恋处理  | 美化劈腿/出轨    | 出轨方必须承担后果，不能洗白             |
| 师生恋    | 权力关系不对等    | 极度敏感，建议避免；如必须写，要在毕业后且无利益关系 |
| 复仇过度   | 报复手段超出合理范围 | 复仇要有度，不能比反派还残忍             |
| 家暴情节   | 展示家暴但无后果   | 必须有反抗、有法律介入、有惩罚            |


### 3. 表现手法灰区


| 灰区内容 | 风险点         | 处理方式                   |
| ---- | ----------- | ---------------------- |
| 恐怖氛围 | 过度惊吓、阴暗画面   | 用悬疑气氛替代直接惊吓，控制恐怖程度     |
| 赌博情节 | 以赌博作为剧情工具   | 赌博者必须付出代价，不能靠赌博翻身      |
| 迷信内容 | 算命、做法等超自然元素 | 可以作为文化元素出现，不能作为解决问题的手段 |
| 奢侈消费 | 过度展示奢华生活    | 适度展示可以，不能以炫富为核心卖点      |
| 医疗情节 | 不专业的医疗处理    | 涉及医疗内容要基本准确，不误导观众      |

## 三、短剧常见踩坑点

### 踩坑清单（按题材分类）

**战神归来类：**

- ❌ 主角随意动用军事力量处理私事
- ❌ 军衔/军装使用不规范
- ❌ 暗示军人可以凌驾于法律之上
- ✅ 军人身份带来的是责任和纪律，不是特权

**霸道总裁类：**

- ❌ 强吻、强行带走、限制人身自由当浪漫
- ❌ 用金钱控制女主的感情和选择
- ❌ 女主毫无独立人格，完全依附男主
- ✅ 尊重对方意愿，双向奔赴，女主有自我

**甜宠类：**

- ❌ 跟踪、偷拍当"深情"
- ❌ 替女主做所有决定当"保护"
- ❌ 吃醋时的暴力/控制行为当"在乎"
- ✅ 健康的恋爱关系：信任、沟通、尊重边界

**家庭伦理类：**

- ❌ 婆媳矛盾中全面丑化一方
- ❌ 将忍耐家暴描写为"美德"
- ❌ 离婚后一方净身出户当"大度"
- ✅ 展示健康的家庭关系处理方式，法律意识

**重生穿越类：**

- ❌ 利用未来知识进行股票交易等违法行为
- ❌ 用现代价值观全盘否定古代
- ❌ 穿越后"开挂"毫无挑战
- ✅ 保留一定历史尊重，主角成长要有代价

**古装宫廷类：**

- ❌ 歪曲真实历史人物形象
- ❌ 美化后宫争斗中的极端手段（下毒害人等）
- ❌ 宣扬皇权至上、奴才思想
- ✅ 可以有宫斗但要有正面角色的坚守

**悬疑探案类：**

- ❌ 详细展示犯罪手法（可被模仿）
- ❌ 美化罪犯或同情化处理
- ❌ 主角使用非法手段获取证据
- ✅ 法律框架内解决问题，正义通过合法途径实现

## 四、正向价值观引导

### 必须传递的正向价值

```
核心正向价值（至少体现2-3个）：
├── 善有善报：好人最终得到好结果
├── 正义必胜：邪不压正，坏人必受惩罚
├── 自强不息：通过正当努力改变命运
├── 家庭和睦：家人之间的爱与支持
├── 诚信守诺：信守承诺、诚实做人
├── 法治精神：在法律框架内解决问题
├── 男女平等：尊重每个人的独立人格
└── 社会责任：个人成功回馈社会
```

### 反面教材处理原则

```
反面内容展示规则：
├── 可以展示恶行（用于激发观众愤怒/同情）
├── 但恶行必须得到惩罚（善恶有报）
├── 不能将恶行美化或合理化
├── 受害者不能"感谢"施害者
├── 主角的复仇手段不能比反派更恶劣
└── 结局必须传递正面信息
```

### 结局价值观要求


| 结局类型 | 价值观要求        |
| ---- | ------------ |
| 大团圆  | 善有善报，坏人受到惩罚  |
| 开放式  | 正面力量占上风，留有希望 |
| 悲剧式  | 引发反思，传递警示意义  |
| 反转式  | 最终反转必须指向正面   |

## 五、平台审核重点

### 各平台关注点差异


| 审核维度   | 重点关注           |
| ------ | -------------- |
| 标题/封面  | 禁止低俗标题党、性暗示封面  |
| 前3集    | 审核最严，决定是否通过    |
| 付费卡点   | 不允许用低俗内容作为付费钩子 |
| 弹幕/评论区 | 内容引导方向是否正面     |
| 整体价值导向 | 全剧传递的核心信息是否正面  |


### 易被退回的问题

```
高频退回原因：
├── 标题含"禁忌""出轨""复仇"等敏感词
├── 封面图过度暴露或暗示
├── 前3集出现过激冲突（打斗、吵架过于激烈）
├── 台词含脏话、人身攻击
├── 反派行为过于真实具体（可被模仿）
├── 结局没有正面收束
├── 涉及真实品牌/机构/人物未授权
└── 背景音乐未获授权
```

## 六、合规自检表

### 逐集自检

写完每集后，逐项检查：


| 序号  | 检查项  | 标准          | 处理     |
| --- | ---- | ----------- | ------ |
| 1   | 红线内容 | 无任何红线内容     | 发现即删除  |
| 2   | 灰区内容 | 已做规避处理      | 调整表现方式 |
| 3   | 台词合规 | 无脏话/歧视/人身攻击 | 修改台词   |
| 4   | 暴力程度 | 无血腥特写/虐待细节  | 弱化处理   |
| 5   | 情感尺度 | 无过度亲密/性暗示   | 调整分寸   |
| 6   | 价值导向 | 本集不传递负面价值   | 补充正面元素 |


### 全剧自检

全剧完成后，整体检查：


| 序号  | 检查项   | 标准           | 处理     |
| --- | ----- | ------------ | ------ |
| 1   | 善恶有报  | 坏人全部受到惩罚     | 补写惩罚情节 |
| 2   | 主角价值观 | 主角行为整体正面     | 调整行为动机 |
| 3   | 女性形象  | 女性角色有独立人格    | 增加独立情节 |
| 4   | 正向结局  | 结局传递正面信息     | 调整结局方向 |
| 5   | 敏感内容  | 无政治/宗教/民族敏感  | 删除相关内容 |
| 6   | 版权问题  | 无未授权品牌/音乐/人物 | 替换为虚构  |
| 7   | 可模仿性  | 无可被模仿的危险行为   | 模糊处理细节 |

## 七、出海合规补充

### 海外市场额外注意事项


| 市场  | 额外合规要求                     |
| --- | -------------------------- |
| 东南亚 | 尊重宗教信仰（伊斯兰教地区尤为敏感）、避免种族歧视  |
| 中东  | 严格着装要求、禁止酒精/猪肉相关内容、宗教敏感度极高 |
| 欧美  | 版权意识强（音乐/品牌）、种族多样性、政治正确    |
| 日韩  | 历史敏感话题、文化尊重、肖像权严格          |
| 拉美  | 避免刻板印象、尊重当地文化习俗            |


### 通用出海合规原则

```
出海内容合规：
├── 删除所有中国特有政治敏感内容
├── 替换中国特有法律/制度描述为通用表达
├── 检查人名/地名是否在目标市场有负面含义
├── 宗教元素处理：中立或删除
├── 肤色/种族：确保多样性和尊重
├── 性别议题：符合目标市场的接受度
└── 文化冲突点：提前排查并调整
```

## 合规处理优先级

```
发现问题时的处理优先级：

P0 - 立即删除（红线内容）：
├── 政治敏感
├── 色情内容
├── 教唆犯罪
└── 未成年相关

P1 - 必须修改（高风险灰区）：
├── 价值观偏差
├── 暴力过度
├── 性骚扰美化
└── 法外开恩

P2 - 建议优化（低风险灰区）：
├── 表现手法偏激
├── 奢侈消费展示
├── 迷信元素
└── 医疗不准确

P3 - 锦上添花（正向引导）：
├── 增加正面价值表达
├── 强化法治元素
├── 补充社会责任感
└── 展示健康关系
```
