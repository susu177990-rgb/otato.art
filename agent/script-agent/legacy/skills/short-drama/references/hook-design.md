# 钩子设计手册

## 核心理念：每集结尾都是一道门

观众看完一集，手指悬在屏幕上——划走还是继续？这个决定在最后10秒做出。钩子的作用就是让那根手指"划不动"。

**钩子公式：** 制造一个观众"必须知道答案"的问题，然后——不给答案。

---

## 5种钩子类型

### 类型一：悬念钩

**一句话定义：** 抛出一个关键疑问，答案留到下一集

**适用场景：** 所有题材通用，最基础也最有效的钩子

**机制：** 建立期待 → 接近答案 → 在揭晓前切断

**子模式：**


| 子模式  | 描述          | 示例                 |
| ---- | ----------- | ------------------ |
| 身份悬念 | 某人真实身份即将揭晓  | 他缓缓摘下面具——          |
| 结果悬念 | 某件事的结果悬而未决  | 验孕棒上的第二条线……        |
| 选择悬念 | 主角面临艰难抉择    | 左边是他，右边是她，她看了看两人—— |
| 来者悬念 | 有人来了但不知道是谁  | 门铃响了。这个时间，会是谁？     |
| 发现悬念 | 发现了什么但观众没看到 | 她打开抽屉，瞳孔骤缩——       |


**剧本示例：**

> △ （特写）她颤抖着打开那个快递盒。
>
> △ 里面不是她订的东西。是一张照片。
>
> △ （特写）照片上的人——她认出来了。
>
> **女主**（画外音，颤抖）："这不可能……你不是已经……"
>
> 【本集完】

**设计要点：**

- 疑问必须和核心剧情相关（不要用无关紧要的悬念骗观众）
- 下一集必须回应这个悬念（不能装作没发生过）
- 回应后可以不完全解答，但必须给出新信息

---

### 类型二：反转钩

**一句话定义：** 在最后一刻抛出颠覆观众预期的信息

**适用场景：** 悬疑探案、重生穿越、古装宫廷、都市情感

**机制：** 观众以为A → 最后一秒揭示其实是B → 切断

**子模式：**


| 子模式  | 描述             | 示例                   |
| ---- | -------------- | -------------------- |
| 身份反转 | 以为是好人实则坏人（或反之） | 她背后传来他的声音——但说的是另一种语言 |
| 局势反转 | 以为赢了实则输了（或反之）  | 合同签完了。但最后一页……        |
| 关系反转 | 以为是A关系实则是B关系   | DNA报告上写着——亲缘关系：父女    |
| 时间反转 | 以为是现在实则是过去/未来  | 镜子里的她，头发全白了          |
| 动机反转 | 以为是这个原因实则另有隐情  | "我从来没有恨过你。我恨的是我自己。"  |


**剧本示例：**

> △ （全景）婚礼现场。宾客掌声雷动。
>
> **主持人**："新郎可以亲吻新娘了。"
>
> △ 新郎掀开头纱。
>
> △ （特写）新郎的手——僵住了。
>
> △ 头纱下面的人——不是她。
>
> 【本集完】

**设计要点：**

- 反转必须有前面的铺垫支撑（不能凭空跳出来）
- 反转的冲击力取决于之前建立的"以为"有多稳固
- 一个好的反转钩能让观众想回去重看前面的剧情

**使用频率：** 全剧不超过5-7次，用多了观众会免疫

---

### 类型三：情绪钩

**一句话定义：** 把情绪推到最高点然后切断，让观众"意犹未尽"

**适用场景：** 甜宠、霸道总裁、都市情感、家庭伦理

**机制：** 情感铺垫 → 情绪升温 → 即将到达顶峰 → 切断

**子模式：**


| 子模式  | 描述           | 示例                |
| ---- | ------------ | ----------------- |
| 甜蜜中断 | 快要亲上了/告白了被打断 | 他凑近，呼吸交缠——手机响了    |
| 心碎中断 | 情感崩溃的最高点切断   | 她转过身，眼泪终于掉下来——    |
| 误会引爆 | 误会刚刚产生的一刻    | 她刚好看到他牵着另一个女人的手   |
| 决绝中断 | 做出痛苦决定的瞬间    | "我们……分开吧。" 他愣住了—— |
| 重逢中断 | 久别重逢但还没相认    | 人群中，她停下脚步。那个背影……  |


**剧本示例：**

> △ （近景）天台。风吹过两人的头发。
>
> **男主**（认真地看着她）："我想了很久，有句话一直没说。"
>
> **女主**（心跳加速）："什么话？"
>
> △ 他深吸一口气，双手捧住她的脸。
>
> **男主**："从遇见你那天起，我——"
>
> △ 楼下突然传来尖叫声。两人同时回头。
>
> △ 楼下——浓烟滚滚。
>
> 【本集完】

**设计要点：**

- 情绪铺垫要足够长（至少半集以上的情感积累）
- 中断的方式要自然合理（不能太刻意）
- 下一集开头要延续这个情绪（不能跳过当没发生）
- 甜宠剧的情绪钩是留存利器，但不能每集都用同一种

---

### 类型四：信息钩

**一句话定义：** 透露一个改变全局的关键信息，但只说一半

**适用场景：** 悬疑探案、重生穿越、古装宫廷、战神归来

**机制：** 发现关键信息 → 开始解读 → 只解读了一部分 → 切断

**子模式：**


| 子模式  | 描述         | 示例                     |
| ---- | ---------- | ---------------------- |
| 证据揭露 | 发现关键证据     | 监控录像里那个身影——穿着和他一模一样的衣服 |
| 秘密泄露 | 无意中听到/看到秘密 | 她躲在门后，听到了不该听的对话        |
| 记忆闪回 | 突然想起被遗忘的记忆 | 她的脑海中闪过一个画面——那天，他不在场   |
| 文件发现 | 发现改变局势的文件  | 保险箱里，一份从未见过的遗嘱         |
| 线索串联 | 散落的线索突然串起来 | 三张照片摆在一起，她突然明白了——      |


**剧本示例：**

> △ （近景）深夜书房。她翻开父亲的旧笔记本。
>
> △ 大部分页面都是普通的工作记录。
>
> △ 直到最后一页。
>
> △ （特写）笔记本最后一页，只有一行字：
>
> "如果有一天你看到这段话，不要相信任何人。包括——"
>
> △ 后面的字被撕掉了。
>
> 【本集完】

**设计要点：**

- 信息必须和主线剧情强相关
- "只说一半"是关键——给够暗示但不给答案
- 不能让观众觉得信息是编剧硬塞进来的
- 信息钩适合推动剧情进入新阶段

---

### 类型五：危机钩

**一句话定义：** 突然出现重大危机或威胁，主角来不及反应

**适用场景：** 末日重生、古装宫廷、悬疑探案、战神归来

**机制：** 平静/胜利的假象 → 突发危机 → 危险逼近 → 切断

**子模式：**


| 子模式 | 描述          | 示例                      |
| --- | ----------- | ----------------------- |
| 突袭  | 敌人突然出现或发动攻击 | 他们刚放松警惕，门被踢开了           |
| 背刺  | 信任的人突然反水    | 她接过他递来的水，喝了一口——他笑了      |
| 暴露  | 主角的秘密/计划被发现 | "你以为我不知道你在做什么？"背后传来冷笑   |
| 倒计时 | 时限将至但还未完成   | 炸弹上的数字：00:00:47。而钥匙还在楼上 |
| 绝境  | 被逼到无路可退     | 前面是悬崖，后面是追兵。她低头看了一眼——   |


**剧本示例：**

> △ （全景）庆功宴。所有人举杯庆祝。
>
> △ 女主笑着和同事碰杯。
>
> △ 手机震动。她低头看了一眼。
>
> △ （特写）消息内容："你老公在三楼开房。拍到了。照片已发你邮箱。"
>
> △ （近景）她的笑容凝固在脸上。
>
> △ 丈夫从对面走过来："老婆，敬你一杯。"
>
> △ 她看着他，手里的酒杯在微微颤抖。
>
> 【本集完】

**设计要点：**

- 危机要突然且出乎意料
- 危机的严重程度要足够大（不能是小事）
- 主角必须来不及反应就被切断（反应留给下一集）
- 不能连续多集都用同类型危机

## 钩子使用策略

### 分阶段钩子配置

```
起势段（前15%集数）：
├── 主要钩子：悬念钩、信息钩
├── 钩子强度：中等
├── 目的：建立好奇心，让观众想看下去
└── 频率：每集结尾都要有

攀升段（15%-45%集数）：
├── 主要钩子：悬念钩、情绪钩、反转钩
├── 钩子强度：中等偏强
├── 目的：维持粘性，制造付费动力
└── 频率：每集结尾都要有，卡点集用最强钩子

风暴段（45%-80%集数）：
├── 主要钩子：反转钩、危机钩、信息钩
├── 钩子强度：强
├── 目的：高密度刺激，让观众停不下来
└── 频率：每集结尾必有，部分集可叠加两种钩子

决战段（最后20%集数）：
├── 主要钩子：危机钩、反转钩、悬念钩
├── 钩子强度：最强
├── 目的：推向终极高潮
└── 频率：除最后一集外每集必有
└── 最后一集：不需要钩子，需要圆满收束
```

### 不同题材的钩子侧重


| 题材   | 主钩子     | 副钩子 | 慎用钩子         |
| ---- | ------- | --- | ------------ |
| 战神归来 | 悬念钩、危机钩 | 信息钩 | 情绪钩（比重不宜过高）  |
| 霸道总裁 | 情绪钩     | 悬念钩 | 危机钩          |
| 甜宠   | 情绪钩     | 悬念钩 | 危机钩（慎用，破坏甜感） |
| 重生穿越 | 信息钩、反转钩 | 悬念钩 | —            |
| 家庭伦理 | 情绪钩、危机钩 | 信息钩 | —            |
| 古装宫廷 | 反转钩、危机钩 | 悬念钩 | —            |
| 悬疑探案 | 信息钩、悬念钩 | 反转钩 | 情绪钩（比重不宜过高）  |
| 末日重生 | 危机钩     | 悬念钩 | 情绪钩          |


### 钩子叠加规则

```
可叠加组合（效果翻倍）：
├── 悬念钩 + 情绪钩：疑问+心动同时存在
├── 反转钩 + 危机钩：反转带来的危机
├── 信息钩 + 悬念钩：信息引出更大疑问
└── 危机钩 + 情绪钩：危险中的情感抉择

不建议叠加：
├── 反转钩 + 反转钩：两个反转互相稀释
├── 信息钩 + 信息钩：信息过载
└── 三种以上钩子叠加：观众处理不过来
```

## 钩子与付费卡点的关系

```
付费卡点 = 最强钩子 + 最高情绪 + 最关键时刻

普通集钩子：
├── 让观众想看下一集（免费继续看）
└── 强度：中等

卡点集钩子：
├── 让观众不得不付费看下一集
├── 强度：最强
└── 必须结合付费卡点策略（参考 paywall-design.md）
```

## 钩子质量自检


| 检查项       | 标准             | 不合格处理     |
| --------- | -------------- | --------- |
| 有未回答的问题吗？ | 至少1个观众迫切想知道的答案 | 增加悬念元素    |
| 情绪到位了吗？   | 观众在结尾时情绪处于高位   | 加强情绪铺垫    |
| 切断时机对吗？   | 在最关键的一刻戛然而止    | 调整切断位置    |
| 和下一集衔接吗？  | 下集开头能自然承接      | 检查衔接逻辑    |
| 钩子和主线相关吗？ | 不是无关紧要的小事      | 替换为核心剧情钩子 |
| 是否重复了？    | 连续3集不用同类型钩子    | 替换钩子类型    |


**致命钩子问题：**

- ❌ 无钩子硬结尾 → 观众没有理由继续看
- ❌ 钩子悬念下集秒解 → 观众觉得被骗（虚假悬念）
- ❌ 连续用同一种钩子 → 观众产生免疫
- ❌ 钩子和剧情无关 → 观众觉得是凑数
- ❌ 钩子给出太多信息 → 观众已经猜到答案，不需要看下集
