# 钩子类型词表与轮换（HOOK_LEXICON）

## 开场钩子（前 3～8 秒可用类型）

- **危机切入**：危险、倒计时、追捕、即将败露
- **关系张力**：暧昧被打断、冷战对峙、被迫同框
- **信息异常**：一条消息、一件物品、一句反常识台词
- **情绪极端**：崩溃边缘、失控边缘、甜虐反转一瞬间

## 集尾卡点（挂尾）类型

与 Development 模板「本集集尾卡点类型」对齐，可从中选用：

- 关系逆转 / 身份揭露 / 打断式亲密 / 误会坐实 / 第三方闯入
- 证据出现 / 承诺破裂 / 秘密被听见一半 / 生死未卜（慎用，需与类型匹配）

## 轮换建议（避免审美疲劳）

- **同型集尾**：不宜 **连续超过 2 集**；第 3 集起必须换型或做复合型（复合时标注主次）。
- **开场与集尾**：尽量避免「同集内」开场与集尾同一套路的无递进重复。
- **信息揭露**：若连续多集只做「揭露」而无关系/情绪代价，观众易疲；穿插「反噬、误用、拒绝相信」等变体。

## 扩展位

项目可在此追加**本类型剧**高频有效钩子，但保持词条简短。
