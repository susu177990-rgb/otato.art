# 声线与辨识度原则（CAST_VOICE）

## 与 `character_reference.md` 的分工

- **具体角色**：姓名、口癖、称谓、禁忌词、双人互动习惯 → `**agent/script-agent/context_assets/character_reference.md`**（权威来源）。
- **本文件**：短剧写作**通用原则**，避免与 `character_reference` 重复逐条人设。

## 短剧辨识度原则

- **一人一口癖或一句式标签**（可随关系阶段微调，但需有演变逻辑）。
- **称谓稳定**：谁称谁什么，除非剧情刻意改口；改口本身是戏。
- **对白作战化**：每句尽量带**试探、遮掩、反击或撒娇式台阶**，减少说明性台词。

## 双人戏

- 权力差要落在**话轮长短、打断、沉默、视线**上，不只落在台词解释上。

## 维护

新增或修改具体角色规则时，只改 `character_reference.md`；此处仅补充「项目级」声线策略。
