# 集尾与连载接戏规则（CLIFFHANGER_RULES）

## 集尾必须满足

- 收在**具体动作或对白刃口**，观众能明确感到「下一秒会发生什么」。
- 与 `Episode Final Script Template` 一致：**不写「下集预告」式空话**（避免「究竟会怎样，敬请期待」类元叙述）。

## 集尾避免

- 纯旁白总结本集（削弱卡点）。
- 无新张力、无新信息的「温柔收尾」除非本集功能为情绪补偿且下集开场已强承诺（需在开发版「本集定位」注明）。

## 下集开篇接戏（约前 3 秒）

- **立刻**回应上集最后一镜/最后半句话：同一时空紧接，或硬切但信息上有直接因果。
- 禁止：下集开头重播上集整段 recap（除非平台强制；若强制，控制在极短一句内）。

## 与 Final 模板「### 集尾」的关系

- 交付版集尾仍须具备**戏剧张力**；若交付版篇幅极短，可合并场次但不可删掉「刃口」。
