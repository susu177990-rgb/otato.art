## 集数未定时的工作方式

1. 先填满**里程碑表**与**阶段任务**。
2. 再选目标总长 **30 / 45 / 60** 之一，将事件压入对应区间；若事件过多，优先合并「同功能戏」或前置揭露；若过少，则在升级段增加「障碍变体」而非硬注水。
