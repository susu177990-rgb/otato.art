# 短剧节奏与信息密度（SHORTFORM_PACING）

## 单集时间轴（秒级节拍）

以下用于**构思与自检**，不必写入交付版正文。


| 区段   | 建议占比            | 任务                                           |
| ---- | --------------- | -------------------------------------------- |
| 开场钩子 | 约 **3～8 秒**     | 冲突、悬念、关系张力或视觉奇观之一，立刻建立「为什么要看下去」              |
| 推进主体 | 约 **40～80 秒**   | 完成本集**一个**主推进（误会加深 / 信息揭露 / 关系质变一步 / 外部压力升级） |
| 集尾卡点 | 约 **最后 8～15 秒** | 收在动作或对白刃口，迫使观众点下一集                           |


## 与 Development 模板字段的对应

- **本集功能**：在主体区完成，且与开场钩子类型一致或递进。
- **本集主钩子**：应用一句话概括「观众本集最爽/最虐/最紧张的点」。
- **本集集尾卡点类型**：须与 `04_HOOK_LEXICON.md` / `05_CLIFFHANGER_RULES.md` 配合，避免连续多集同型（见轮换建议）。

## 信息密度上限（单集）

- **新硬信息点**（新身份线索、新规则、新关系事实）：建议 **0～2 个**；超过需确认是否拆集。
- **情绪主色**：每集 **1 个**主情绪（可带次要波动），避免 90 秒内情绪散焦。

## 连载节奏（中弧）

- 每 **5～10 集**安排一次**小高潮或关系/信息升级**，避免长线平铺。
- 里程碑与总长伸缩见 `03_SERIES_BIBLE.md`。

## 幕级节奏（与 1～2 分钟 / ≥8 幕对齐）

- 单集 **8～12 幕**时，平均每幕约 **10～15 秒**量级，与「单幕 ≤15s」模板一致；开场钩子优先落在 **幕 1～2**，集尾刃口落在**最后一幕**。
- 若某场戏情绪重、对白略长，仍须拆成**多幕**（用镜头/动作/停顿切分），避免单幕超时导致下游 AI 视频难拆。
