# 创作方向包：女频 BL 短剧

本项目选择的创作方向为 **女频 BL 短剧**。这是当前 oTATo Art Script Agent 已完整开发并验证的默认方向包。

## 方向定位

- 面向海外女性向 BL 商业短剧，不是泛剧本、泛网剧或电影剧本默认模式。
- 核心关系通常围绕双男主展开，优先处理禁忌感、拉扯感、占有欲、误会、救赎、身份差与情绪反转。
- 默认采用微短剧工业化节奏：常见 30~60 集、单集约 1~2 分钟、每集强钩子、集尾可接戏。
- 默认角色语境为海外本地化，除非主创明确要求中国 / 华语 / 华裔语境。
- STAGE 7 默认遵守英语对白 + 句末中文翻译，并服从项目级英语 Locale 简报。

## 执行纪律

- 后续立项、人物、事件、设定集、分集大纲和分集剧本都必须围绕本方向组织，不得在未获主创明确确认时切成其他类型。
- 通用编剧结构只负责阶段和产物纪律；本方向包负责市场、受众、关系动力、短剧商业节奏和对白默认值。
- 若用户提出非 BL、非短剧或非海外女性向需求，先指出这会偏离当前项目创作方向，再等待主创明确要求后才调整。
