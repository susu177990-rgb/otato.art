# Shared Module：英语对白与 Locale

本模块只描述英语对白和本地化交付方法，不绑定 BL 或任何特定题材。

## 适用边界

- 仅当项目明确要求英语对白、英语市场、本地化改写或 Locale 简报时加载。
- 不自动把所有项目改成英语对白。
- 不自动决定角色国籍、族裔、城市或文化背景。

## 英语对白规则

- 先确认目标英语市场和角色语境，再决定语气、俚语密度、文化引用和礼貌程度。
- 台词应像真人在该场景下说话，避免直译中文语序。
- 避免 Chinglish、解释腔、书面腔和 AI 式长句。
- 情绪信息优先通过动作、停顿、潜台词和回应节奏表达。
- 如果项目要求中英双语交付，中文翻译应服务理解，不反向污染英语台词。

## Locale 简报

- 明确地区、年龄层、阶层、职业、亲密关系表达边界和禁用表达。
- 后续对白必须服从项目级 Locale 简报。
