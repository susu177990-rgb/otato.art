# Shared Module：改编通用工作流

本模块只描述从原文到目标媒介的通用改编流程，不绑定短剧、BL 或英语对白。

## 原文分析

- 识别原作核心吸引力：主角欲望、主要关系、世界观承诺、类型卖点、情绪回报。
- 区分必须保留、可压缩、可合并、可重排和可删除的内容。
- 标出原作中不适合目标媒介或目标市场的风险点。

## 改编讨论

- 先确定目标媒介和体量，再决定删留策略。
- 不把原作情节机械搬运到新媒介。
- 不在目标方向未确认时硬套某一套类型模板。

## 改编规划

- 输出创作思路确认书时，要写清保留核心、重排逻辑、人物合并、节奏策略、目标受众和风险。
- 若信息不足，列出待确认项和 2~3 条可执行候选路径。
