# Shared Module：短剧节奏与钩子

本模块只描述短剧 / 微短剧 / 竖屏剧的通用工业方法，不绑定 BL、女性向、英语对白或任何特定关系结构。

## 适用边界

- 仅当项目明确选择短剧、微短剧、竖屏剧、高密度分集内容时加载。
- 不自动决定题材、CP 关系、市场、角色身份或对白语言。
- 如果项目不是短剧，不应套用本模块。

## 通用短剧规则

- 单集必须有清晰事件目标、情绪推进和集尾接戏点。
- 前几集快速建立人物困境、欲望、阻碍和观看承诺。
- 中段用误会、选择、代价、反转、关系变化或外部压力维持推进。
- 付费点应落在“观众已经理解冲突，但关键答案尚未释放”的位置。
- 集尾钩子必须服务下一集行动，而不是只写口号式悬念。
- 体量、单集时长和语言必须服从项目级设置。

## 禁区

- 不把短剧默认等同于 BL。
- 不把短剧默认等同于海外女性向。
- 不把短剧默认等同于英语对白。
