# 示例：游艇侧栏首帧重构

本示例只描述输入摘要和期望输出形态，不包含真实图片文件。

## 输入摘要

### 原实拍首帧

- 素材索引：`SRC_FIRST_FRAME`
- 画幅：竖幅，接近 `9:16`
- 画面：低机位仰拍女主背侧，女主身体靠近镜头，双手向右上方抓住影棚横杆。
- 可保留：人物姿态、低机位仰拍、双手抓握动作、人物前景大比例、背侧表演状态。
- 必须废弃：影棚天花板、灰幕、室内结构、原始横杆的场景属性、原始背景深度。

### 目标场景资产宫格图

- 素材索引：`SCENE_GRID`
- 内容：豪华游艇外观、游艇侧边、金属栏杆、甲板、船体、海面、天空、海平线。

### 带位置标识的目标场景图

- 素材索引：`MARKED_SCENE_GRID`
- 标注：红点代表女主身体中心，黄色点代表双手抓握栏杆位置，蓝色箭头代表低机位仰拍方向。

### 角色图

- 素材索引：`CHARACTER_女主`
- 绑定名称：女主

### 用户意图

把原实拍中女主抓横杆的动作，转绘成女主抓住豪华游艇侧边金属栏杆，身体像挂在游艇一侧，镜头从下往上拍，背景看到白色船体、栏杆、海面和天空。

## 镜头重构师期望输出形态

【启用模式】
Target Set Reconstruction Mode

【保留原画面】
保留女主低机位仰拍、身体靠近镜头、双手向右上方抓握、人物前景大比例、背侧表演姿态和竖幅构图。

【废弃原画面】
废弃影棚天花板、灰幕、室内结构、原始横杆的场景属性和原始背景深度。

【位置】
人物位置以 `MARKED_SCENE_GRID` 中红点标注的位置为准；红点代表女主身体中心。双手必须对齐黄色接触点，抓住游艇侧边栏杆。

【比例】
9:16（按照原实拍首帧图的竖幅比例确定）。

【目标场景重构】
目标空间重构为豪华游艇侧边。女主在前景，金属栏杆穿过双手抓握位置，白色船体位于她身后和下方，海面与天空作为远景。

【关键接触关系】
双手抓住金属栏杆，栏杆与手掌形成真实遮挡。身体重心靠近游艇侧边，由栏杆和船体关系提供支撑，人物不能悬浮。

【前中后景】
前景是女主身体和手臂；中景是游艇栏杆、白色船体和甲板边缘；远景是海面、天空和海平线。

【禁止错误】
不要出现影棚天花板、灰幕或室内结构。不要让手抓空气。不要让游艇变成平面背景。不要改变竖幅比例。

【状态】
ready

【使用素材】
- 结构参考：`SRC_FIRST_FRAME`
- 场景参考：`SCENE_GRID`
- 位置参考：`MARKED_SCENE_GRID`
- 角色参考：`CHARACTER_女主`
- 道具参考：无强制道具

【比例】
9:16（按照原实拍首帧图的比例确定）

【最终生图 Prompt】
启用 Target Set Reconstruction Mode。以 `SRC_FIRST_FRAME` 锁定女主姿态、低机位仰拍、双手抓握动作、人物前景大比例和竖幅构图；不要使用原始影棚背景深度作为最终空间。以 `MARKED_SCENE_GRID` 的红点作为女主身体中心位置，以黄色点作为双手抓握栏杆接触点。目标场景根据 `SCENE_GRID` 重构为豪华游艇侧边空间：女主抓住游艇一侧金属栏杆，身体像倚靠或悬挂在游艇侧边，白色船体位于她身后和下方，甲板边缘沿画面透视延展，远景是蓝色海面、天空和海平线。栏杆必须穿过双手抓握位置并形成真实遮挡，人物不能悬浮。角色外观服从 `CHARACTER_女主`。

【Negative Prompt】
影棚天花板，灰幕，室内结构，原始横杆残留，平面背景贴图，手抓空气，栏杆错过手掌，人物悬浮，比例错误，横幅构图，前中后景混乱，肢体变形，塑料感皮肤，廉价 3D 感。
